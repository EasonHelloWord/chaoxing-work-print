# 学习通作业题目答案打印整理

一个用于学习通/超星作业页面的浏览器脚本。它会在作业详情页、作业列表页右侧显示悬浮面板，帮助你把作业题目、选项、答案、解析整理成适合打印、保存 PDF、复制或批量导出的文件。

适合这些场景：

- 考前整理已完成作业的题目和答案。
- 批量保存课程作业为 HTML、Markdown、TOML。
- 将多个作业合并到一个打印页，保存为 PDF。
- 对未完成、待批阅、超时/过期等无答案作业导出题目和选项。
- 打印双面资料时，让不同作业尽量从新的纸张正面开始。

## 主要功能

- 单个作业导出 PDF，复制题目文本，导出 HTML / Markdown / TOML。
- 批量读取课程作业列表，按当前筛选和勾选列表导出。
- 批量 PDF 每篇作业自动新开页面。
- 批量文本格式会打包为 ZIP 下载。
- 支持有答案、待批阅、未完成、超时/过期、其他状态筛选。
- 无答案作业自动标注“无答案”，并且不会输出答案或解析。
- 支持按题型筛选：单选、多选、判断、填空、简答、论述、计算、其他题型。
- 支持跳过同名作业、生成导出清单、显示/隐藏解析。
- 支持图片导出为在线链接或尽量转换为 base64。
- 支持按时间或名称正序/倒序排序，也可以拖拽手动微调导出顺序。
- 悬浮窗可收起，减少对学习通页面的遮挡。

## 安装

推荐使用 Tampermonkey、Violentmonkey 等浏览器脚本管理器。

1. 安装浏览器脚本管理器。
2. 在 Greasy Fork 安装本脚本，或从 GitHub 打开 `chaoxing-work-print.user.js` 后点击 Raw 安装。
3. 进入学习通课程作业列表页或作业详情页。
4. 页面右侧出现“学习通作业题目答案打印整理”悬浮面板后即可使用。

## 使用说明

### 单个作业

进入作业详情页或作答页后，悬浮窗会显示已识别题目数量。

- `导出PDF`：打开整理后的打印页面，并唤起浏览器打印窗口。选择“保存为 PDF”即可保存。
- `复制文本`：复制题目、选项、答案和解析文本。
- 格式下拉：选择 `HTML` / `Markdown` / `TOML` 后点击 `导出文件`。
- 齿轮按钮：展开导出设置，可控制题型、解析、图片格式等选项。

### 批量导出

进入课程作业列表页或课程首页后，悬浮窗会读取当前课程的作业列表。

- `导出PDF`：按当前筛选和勾选列表生成批量打印页。
- 格式下拉：选择 `HTML` / `Markdown` / `TOML` 后点击 `导出`，会按当前设置打包为 ZIP。
- `刷新`：重新读取作业列表。
- `全选` / `全不选`：快速调整将要导出的作业。
- 作业列表：点击可勾选/取消，拖拽可调整导出顺序。
- 齿轮按钮：展开导出范围、导出选项、题型筛选和图片设置。

## 导出设置

导出范围：

- 有答案
- 待批阅
- 未完成
- 超时/过期
- 其他状态

导出选项：

- 显示解析
- 跳过同名
- 无答案标注
- 导出清单
- 打印友好
- 排序：自动、时间正序、时间倒序、名称正序、名称倒序

题型筛选：

- 单选题
- 多选题
- 判断题
- 填空题
- 简答题
- 论述题
- 计算题
- 其他题型

图片设置：

- 在线链接：文件更小，图片仍从学习通图片地址加载。
- base64：尽量把图片写入导出文件，方便离线保存；部分跨域或受限图片可能仍保留原链接。

## 说明与限制

- PDF 导出使用浏览器打印功能，不会直接生成 PDF 文件；请在打印窗口中选择“保存为 PDF”。
- 批量导出需要当前账号有权限访问对应作业。
- 未完成、未交、待批阅等作业通常没有标准答案，脚本只导出题目和选项，并标注“无答案”。
- 学习通页面结构可能变化，如果遇到识别失败，可以在 GitHub Issues 反馈页面地址、截图和复现步骤。

## 隐私与数据

脚本在浏览器本地运行，不会主动上传题目、答案或账号信息到第三方服务器。

批量导出时，脚本会在当前登录态下请求学习通页面，以便读取作业列表和作业详情。导出的 ZIP、PDF、HTML、Markdown、TOML 文件只会保存在你的本地设备。

不要把本地 HAR 抓包、课程页面 HTML、导出的 ZIP/PDF 直接提交到公开仓库，这些文件可能包含课程、账号、作业参数或其他隐私信息。
