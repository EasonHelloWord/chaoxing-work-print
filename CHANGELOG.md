# Changelog

## 1.5.2

- 调整导出操作区，PDF 作为第一主按钮。
- 移除单个作业页“打印版”按钮，统一使用“导出PDF”。
- HTML、Markdown、TOML 改为通过格式下拉选择后导出，减少悬浮窗占用空间。

## 1.5.1

- 批量作业排序改为作业级排序，支持自动、时间正序/倒序、名称正序/倒序。
- 批量篇目列表默认支持拖拽微调，不再显示“手动排序”选项。
- 移除小题排序设置，仅保留题型筛选。
- 优化悬浮窗按钮、设置区和列表的视觉样式。

## 1.5.0

- 增加 Markdown 和 TOML 文本格式导出，单个作业可直接下载，批量导出会打包为 ZIP。
- 增加题型筛选，单个作业和批量导出共用同一套题型选项。
- 增加题目排序设置，可保持原顺序或按题型自动分组。
- 增加批量篇目手动排序，启用后可拖拽篇目列表决定导出顺序。
- 增加图片模式设置，可选择在线链接或尽量转换为 base64。
- 增加悬浮窗收起按钮。

## 1.4.0

- 批量 PDF 打印页改为每篇作业独立开新页，便于按作业分隔保存或打印。
- 增加“打印友好”导出选项，可在估算为奇数页的作业后自动补一页空白页，减少双面打印串页。

## 1.3.0

- 增加批量导出 PDF 打印页。
- 增加单个作业导出 PDF 按钮。
- 将批量筛选设置分为“导出范围”和“导出选项”。
- “显示解析”设置会影响打印和 PDF 页面默认显示状态。

## 1.2.4

- 批量导出未交作业时，支持从 `task` 中间页继续解析跳转后的作答页。
- 详情抓取会尝试多个候选 URL，减少未交/未做作业误报“未识别到题目”。

## 1.2.2

- 将面板复选框改为自绘控件，避免被学习通页面样式覆盖。

## 1.2.0

- 增加齿轮设置面板。
- 增加将要导出的篇目列表。
- 增加全选、全不选和刷新。
- 增加“显示解析”批量导出选项。

## 1.1.0

- 批量导出增加状态筛选。
- 无答案作业文件名和标题自动标注“无答案”。
- 支持跳过同名作业和生成导出清单。

## 1.0.x

- 支持作业详情页整理打印、复制文本和下载 HTML。
- 支持作业列表批量导出。
- 支持未批阅、未做、未交等无答案作业导出题目和选项。
