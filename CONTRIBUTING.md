# Contributing

欢迎提交 issue 和 pull request。为了方便维护，请尽量遵守以下约定。

## 提交问题

请说明：

- 页面类型：作业详情页、作答页、课程作业列表或课程首页。
- 作业状态：已完成、待批阅、未交、未做、超时等。
- 实际现象和期望结果。
- 浏览器和脚本管理器版本。

不要在公开 issue 中上传包含个人信息、课程信息、Cookie、URL 参数或答案内容的 HAR 文件。

## 开发检查

修改脚本后运行：

```bash
npm run check
```

## 代码风格

- 保持单文件 userscript，除非确实需要拆分。
- 新功能优先保持本地运行，不引入外部依赖。
- 不上传课程数据、HAR 抓包、导出文件或个人隐私内容。
